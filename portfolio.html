📖 내용

개요

이 프로젝트는 MZ세대 대상으로 한 맞춤형 향수 추천 쇼핑몰 입니다.
이 프로젝트의 목표는 개성과 취향을 중시하는 MZ 세대가
자신의 스타일과 성격에 맞는 향수를 쉽게 찾을 수 있도록 돕는 것입니다.

프로젝트 목표

▶ 맞춤형 향수 추천을 통해 사용자의 선택의 어려움을 줄이기
▶ 향수 입문 자들이 쉽게 접근할 수 있는 환경 조성
▶ MZ 세대를 타겟 으로 한 효과적인 마케팅 전략 수립

주요 기능

▶ 옷 스타일에 맞는 향수 추천 테스트
  ▪ 사용자가 원하는 패션 스타일을 3가지 선택하면
해당 스타일에 어울리는 향수를 추천
▶ 향수 MBTI 테스트
  ▪ MBTI 성격 유형과 유사한 테스트를 통해 사용자의 성격과
    취향을 분석, 맞춤형 향수 추천

개발 인원 : 혼자

▫ 메인 페이지 : 최신 물품 등록, 카테고리 물품, 인기 물품, 추천 물품
  네비게이션 바 반응형 웹 페이지로 구현

▫ 상세 페이지 : 메인 페이지에서 물품 클릭 시 이동

▫ 마이 페이지 : 상세 페이지에서 하트 클릭 시 상품 찜할 수 있음
  엑스 클릭하면 찜한 목록에서 삭제 됨
  내가 등록한 물품 목록 볼 수 있고,   물품 등록 페이지로 가는 링크 있음
▫ 물품 등록 페이지 : 상품 이미지와 글 입력 하여 등록할 수 있음

▫ 로그인 / 회원가입 페이지 : 데이터베이스 연동

  다시 입력해주세요..!!” 뜨면서 로그인 페이지에 머물도록 설정

▫ 로그아웃 : 네비게이션 바에 LogOut 클릭 시 로그아웃 됨
